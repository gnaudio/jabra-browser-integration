Jabra Browser Integration (unstable BETA)

Call control and more from within a browser


Overview

This software from Jabra helps developers to make solutions, where basic headset call control can be used from within a browser app using JavaScript.

Documentation

  • Documentation
  • Typescript API definition/documentation

Development tools

  • Library call control test - simple testpage to try out call control in library
  • Library api test - testpage that allows detailed testing of individual API calls
  • Playback with autoselection - sample showing autoselection of jabra device with simple audio playback
  • WebRTC softphone demo supports mute/unmute/end-call from a Jabra device + jabra device autoselection

Miscellaneous
